# Registro de uso de IA

Se utilizaron herramientas de asistencia (ChatGPT) para generar plantillas de código, estructura del repo y sugerencias de diseño. Prompts principales:

- "Crea un dataset sintético de 1 año para un producto 'Empanada' con features precio, promoción, temperatura, día de semana y ventas."
- "Genera código para entrenar un modelo de regresión lineal y guardar el modelo."

Se revisaron y adaptaron manualmente todos los fragmentos generados.
